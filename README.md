## Comenzando 🚀

Lo primero que debe hacer es clonar este repositorio mediante el comando:

```sh
git clone https://github.com/AdrianGilGamboa/UPOSPORTS.git
```

Mira **Despliegue** para conocer como desplegar el proyecto.
